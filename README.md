## 💾 `.env` File (Main Config)
#### `.env` config table
| `.env` varriable | Description | Type |
|---|---|:---:|:---:|:---:|:---:|
| `TOKEN` | Discord bot token | String |
| `MONGO` | MongoDB | Url |
| `SECRET` | Discord bot secret | String |
